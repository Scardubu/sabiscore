## Phase 3 — Per-League Calibration & Diversity Diagnostics

**Files changed:**  
- `backend/src/models/calibration.py` — new file (501 lines)  
- `scripts/optuna_tune_ensemble.py` — steps 6.5 and 6.6 added to `train_one_league()`

### Calibration (`calibration.py`)

**Method selection rule:**

| n_training_rows | Method |
|---|---|
| ≥ 2000 | Isotonic regression (one per class, binary one-vs-rest) |
| < 2000 | Platt scaling (LogisticRegression on raw probabilities) |
| forced | Temperature scaling (grid search T ∈ [0.1, 10], NLL minimised) |

**Key functions:**
- `select_calibration_method(n, *, isotonic_min_rows=2000, force=None)` → `CalibrationMethodName`
- `fit_calibrator(method, y_true, y_proba)` → fitted calibrator object(s)
- `apply_calibrator(method, calibrators, y_proba)` → renormalised `[n, 3]` array
- `compute_ece(y_true, y_proba, n_bins=10)` → per-class + mean ECE dict
- `run_league_calibration(league, y_train, proba_train, y_val, proba_val)` → `FittedCalibrator`
- `write_calibration_report(fc, output_dir)` → `calibration_report_{league}.json`

**`FittedCalibrator` dataclass fields:** `method`, `league`, `n_training_rows`, `calibrators`, `ece_before`, `ece_after`, `draw_f1_before`, `draw_f1_after`, `selection_rationale`

### Diversity Diagnostics (`calibration.py`)

**`EnsembleDiversityDiagnostics`** (threshold default: `ENSEMBLE_CORRELATION_PRUNE_THRESHOLD = 0.92`, env-overridable):

- `pairwise_correlation(models, X)` — Pearson r of flattened `[n×3]` probability vectors
- `flag_redundant(names, corr)` — members with any off-diagonal r > threshold
- `prune_if_safe(models, X_holdout, y_holdout, names, corr, flagged)` — removes flagged members only when draw-F1 is non-degrading after removal; rationale recorded per member
- `run(league, models, X_holdout, y_holdout)` → `(retained_models, DiversityReport)`
- `write_diversity_report(report, output_dir)` → `diversity_report_{league}.json`

### Integration into `optuna_tune_ensemble.py`

After step 6 (holdout evaluation), `train_one_league()` now executes:

**Step 6.5** — diversity diagnostics: runs `EnsembleDiversityDiagnostics.run()` on the three base models; if any member is pruned, meta-learner is retrained on the reduced set and metrics are re-evaluated.

**Step 6.6** — calibration: calls `run_league_calibration()` using the meta-model's raw probability output on the meta-train and holdout splits; writes `calibration_report_{league}.json`.

The `.pkl` artifact now includes:
```python
{
    "calibrator": FittedCalibrator,          # ← new
    "model_metadata": {
        "diversity": { ... },                # ← new: mean_off_diag, pruned/retained
        "calibration": { ... },              # ← new: method, ece deltas, draw_f1_delta
        ...
    }
}
```
